## Hourly Bike Rental Demand Forecasting

Predicts hourly bike rental demand using temporal and weather features with a diagnosis-driven ML workflow. Models progress from linear baselines to a tuned Gradient Boosting regressor based on residual analysis and empirical validation.

**Dataset:** Bike Sharing Dataset (Hourly), UCI Machine Learning Repository  
https://archive.ics.uci.edu/ml/datasets/bike+sharing+dataset

**Final Model:** Tuned Gradient Boosting Regressor  
**Test RMSE:** ~77 rentals per hour
